# Backend Engineering Portfolio — Hirthikk Kumar G S

A collection of backend projects built to demonstrate real-world engineering skills.
Stack progression: Python → Flask → PostgreSQL → Redis → Docker → AWS

## Projects

### P1 — CLI Log Analyzer
Real-time log file analyzer with spike detection and rich terminal output.
**Stack:** Python, rich, pytest
**Folder:** [CLI LOG ANALYSER](./CLI%20LOG%20ANALYSER)

### P2 — URL Shortener API
REST API that shortens URLs with Redis caching and click analytics.
**Stack:** Flask, PostgreSQL, Redis, Docker
**Folder:** [URL SHORTENER](./URL%20SHORTENER)

### P3 — Job Tracker API
Authenticated REST API to track job applications with JWT auth and role-based access.
**Stack:** Flask, PostgreSQL, Flask-JWT-Extended, bcrypt, Docker
**Folder:** [job-tracker-api](./job-tracker-api)

### P4 — Rate Limiter API
Implements four rate limiting algorithms from scratch using Flask middleware and Redis.
**Stack:** Flask, Redis, Docker
**Folder:** [rate-limiter-api](./rate-limiter-api)
